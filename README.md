#To-Do List App

A  professional **To-Do List Web App** built with **HTML, CSS, and JavaScript**

---

## 🚀 Features

- ✅ Add, Edit, Delete, Complete tasks
- 📅 Set task Deadlines
- 🚦 Set task Priority (Low, Medium, High)
- 🔍 Filter tasks by Priority
- 💾 Persistent data via LocalStorage

---

## 🛠️ Tech Stack

- **HTML5**
- **CSS3**
- **JavaScript (Vanilla)**
